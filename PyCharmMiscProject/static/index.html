Diet Plan API

ALLERGY ALERT: Patient has SEVERE shellfish allergy. All seafood recommendations exclude shellfish.

This API provides access to a personalized diet plan with allergy alerts for Michael Rodriguez.

Available Endpoints

Get Full Diet Plan

GET /diet-plan

Returns the complete diet plan including all days, meals, and nutritional information.

Try it

Get Daily Plan

GET /diet-plan/daily/{day}

Returns a specific day's diet plan. Replace {day} with a number (1-7).

Example: Day 1 | Day 2

Get Specific Meal

GET /diet-plan/meal/{day}/{meal_name}

Returns a specific meal from a day's diet plan.

Example: Day 1 Breakfast | Day 2 Lunch

Get Allergy Information

GET /diet-plan/allergies

Returns allergy information and alerts from the diet plan.
